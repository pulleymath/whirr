# 코드맵 (CODEMAP)

에이전트와 기여자가 **어디를 고치면 되는지** 빠르게 찾기 위한 거친 지도입니다. 파일 단위 나열은 하지 않습니다.

## `src/app`

Next.js App Router의 루트입니다. 전역 스타일·루트 레이아웃, 홈, 세션 상세(동적 세그먼트) 등 **페이지와 레이아웃**을 둡니다. UI를 새로 넣거나 라우트를 바꿀 때는 여기서 진입합니다.

## `src/app/api`

**Route Handler**(서버 API) 전용입니다. STT 연결·일괄 전사·요약 생성처럼 클라이언트가 호출하는 서버 경계를 수정할 때 이 트리를 봅니다.

## `src/components`

**화면 단위 React 컴포넌트**입니다. 홈 콘텐츠, 녹음기, 세션 목록·상세·편집 모달, 전사 보기 등 사용자에게 보이는 블록이 모여 있습니다. 마크업·상호작용·페이지 조합을 손볼 때는 대개 이 디렉터리입니다.

## `src/hooks`

**클라이언트 상태와 브라우저 API**를 감싼 훅입니다. 마이크 녹음·실시간 전사 파이프라인처럼 여러 컴포넌트에서 재사용되는 동작을 여기 둡니다. 녹음·전사 흐름을 바꿀 때 우선 확인합니다.

## `src/lib` (DB·도메인 유틸)

브라우저 **IndexedDB 등 로컬 저장**, 오디오 보조, 세션 텍스트 조립·날짜 그룹·미리보기 등 **순수에 가까운 도메인 로직**이 루트 `lib/`에 모여 있습니다. 데이터 모델이나 저장 방식을 바꿀 때 이 트리와 인접 모듈을 함께 본다.

## `src/lib/stt`

**음성-텍스트(STT) 백엔드 연동**입니다. 프로바이더별 구현, 공개 타입·팩토리, 사용자에게 보일 오류 문구 매핑이 여기에 모입니다. 전사 품질·연결 방식·에러 처리를 바꿀 때 이 폴더를 수정합니다.

## `src/lib/api`

**앱 내부 API Route에서만 쓰는 서버 측 보조 로직**입니다. STT 토큰 발급과 함께 쓰는 레이트 리밋 등, `app/api` 핸들러가 의존하는 공유 코드를 둡니다.

## `src/lib/glossary`

**요약용 전역 용어·세션 컨텍스트 타입과 클라이언트 Context**입니다. 브라우저 저장소에 두고 녹음·설정 UI와 요약 요청 맥락을 맞출 때 이 트리를 봅니다.

## 기타 최상위

- **`src/types`**: 공용 타입을 모을 자리(비어 있을 수 있음).
- **`src/__tests__`**: 앱 전역·계약 수준 테스트가 있으면 여기에 둡니다.
