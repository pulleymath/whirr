# 문제 해결 (Troubleshooting)

증상 → 가능한 원인 → 확인할 것. 스택 트레이스는 포함하지 않는다.

## 빠른 참조

| 증상                            | 가능한 원인                                                           | 확인할 것                                                                                                                                                            |
| ------------------------------- | --------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| STT 연결 실패, 토큰 오류        | API 키 누락·오타, 로컬/프로덕션 환경 변수 불일치, 호출 한도 초과(429) | [SETUP](./SETUP.md) 환경 변수·로컬 실행, [SECURITY](./SECURITY.md) 키 보관·노출 여부, [DEPLOY](./DEPLOY.md) 프로덕션 env, [DECISIONS](./DECISIONS.md) 외부 API·정책  |
| 실시간 전사가 끊기거나 멈춤     | WSS 비정상 종료, **세션 시간(약 60분) 만료**, 네트워크 순단           | 자동 재연결 안내 후에도 지속되면 **녹음 중지 후 다시 시작**. 일괄 전사는 **«다시 시도»**로 동일 녹음 Blob 재전송 가능 — [DECISIONS](./DECISIONS.md) 재시도·한도 정책 |
| 마이크 권한 거부                | 브라우저 권한 차단, 비보안 컨텍스트(HTTP 등)                          | 브라우저 사이트 설정에서 마이크 허용, HTTPS 또는 브라우저가 허용하는 로컬 호스트인지 확인                                                                            |
| IndexedDB·세션 목록이 비어 있음 | 데이터가 클라이언트에만 있음, 다른 브라우저·프로필·기기               | [DECISIONS](./DECISIONS.md)·[ARCHITECTURE](./ARCHITECTURE.md) 저장 위치·경계, 동일 프로필·브라우저인지 확인                                                          |
| 빌드 실패                       | Node 버전 불일치, 린트·테스트 실패                                    | [SETUP](./SETUP.md) 권장 Node 버전, `lint`/`test` 실행 절차                                                                                                          |

## 요약 불릿

- **STT·토큰**: 키가 올바른 환경(로컬 `.env` vs 배포 플랫폼)에 있는지, 429면 호출 제한·재시도 간격을 [DEPLOY](./DEPLOY.md)·[DECISIONS](./DECISIONS.md)와 맞춰 본다.
- **마이크**: 권한 프롬프트·사이트별 설정; 로컬 개발은 [SETUP](./SETUP.md)과 브라우저 정책을 함께 본다.
- **세션·IndexedDB**: 서버 동기화가 없다면 기기·프로필이 바뀌면 목록이 달라진다 — [ARCHITECTURE](./ARCHITECTURE.md)·[DECISIONS](./DECISIONS.md).
- **빌드**: Node 버전 정렬 후 린트·테스트를 돌려 실패 원인을 좁힌다 — [SETUP](./SETUP.md).
