# 이슈 트래커 (Status)

현재 프로젝트의 작업 상태를 파악하기 위한 문서입니다. 각 작업의 진행 상황을 업데이트해주세요.

## 진행 상태 요약

| 상태                     | 의미                         |
| ------------------------ | ---------------------------- |
| 📝 할 일 (To Do)         | 아직 시작하지 않은 작업      |
| 🔄 진행 중 (In Progress) | 현재 작업 중인 이슈          |
| ✅ 완료 (Done)           | 개발 및 테스트가 완료된 작업 |
