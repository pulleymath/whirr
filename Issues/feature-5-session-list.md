# Feature 5: 과거 세션 목록 및 상세 보기 UI

## 1. 개요

IndexedDB에 저장된 과거 녹음 세션들을 날짜별로 그룹화하여 목록으로 보여주고, 클릭 시 전체 텍스트를 확인할 수 있는 UI를 구현합니다.

## 2. 상세 기획 (Detailed Plan)

- **세션 데이터 불러오기 및 그룹화**: `src/lib/db.ts` 또는 유틸리티 파일
  - IndexedDB에서 전체 세션 데이터를 불러오는 로직 작성.
  - 불러온 세션 데이터를 `createdAt` 기준 날짜별로 그룹화하고, 최신순으로 정렬하는 유틸리티 함수 구현.
- **세션 목록 UI 컴포넌트 구현**: `src/components/session-list.tsx`
  - 날짜별로 그룹화된 리스트 렌더링.
  - 각 항목에 생성 시각(예: 오후 2:30)과 텍스트 미리보기(앞 N자) 표시.
  - 최신 순으로 정렬되어 표시.
- **세션 상세 보기 UI 구현**: `src/app/sessions/[id]/page.tsx` 또는 모달 컴포넌트
  - 세션 항목 클릭 시, 해당 세션의 전체 전사 텍스트를 보여주는 상세 페이지 또는 모달 구현.
  - 뒤로가기 또는 닫기 버튼 추가.
- **메인 화면 연동**: `src/app/page.tsx`
  - 메인 화면 하단 또는 사이드바에 세션 목록 컴포넌트 배치.
  - 새로운 세션이 저장될 때 목록이 자동으로 갱신되도록 상태 관리 연동.

## 3. 완료 조건 (Done Criteria)

- [ ] IndexedDB에 저장된 세션들이 날짜별로 정확히 그룹화되어 최신순으로 표시된다.
- [ ] 세션 항목에 텍스트 미리보기가 올바르게 나타난다.
- [ ] 특정 세션을 클릭했을 때 전체 전사 텍스트를 확인할 수 있는 상세 화면(또는 모달)이 열린다.
- [ ] 새로운 녹음이 완료되어 세션이 추가되면, 페이지 새로고침 없이 세션 목록에 즉시 반영된다.
