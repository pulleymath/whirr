# Feature 17: 홈·세션 상세 UI 개선 (녹음 컨트롤, 탭, 버튼 일관성)

## 범위

### 홈 화면

1. 녹음 시작 / 녹음 종료 버튼을 빨간 동그란 원(시작)과 둥근 사각형(중지) 형태로 표시하고, 상태 전환 시 원↔사각형이 부드럽게 바뀌는 애니메이션을 적용한다.
2. 홈에서는 회의록 탭을 제거하고 스크립트만 표시한다. (세션 상세는 회의록·스크립트 탭 유지)
3. ~~마이크 권한에 따른 상단 안내 문구 조건부 표시~~ **이번 작업에서 제외한다.**

### 세션 상세 화면

1. "뒤로" 버튼과 오디오 미리듣기(`<audio controls>`)를 제거한다. (오디오 다운로드 버튼은 유지할지: 이슈에서 미리듣기만 명시 — 미리듣기만 제거, 다운로드는 유지)
2. 복사, 다시 생성, 저장 관련 액션을 `lucide-react` 아이콘을 사용한 버튼으로 표시한다. (아이콘만 또는 아이콘+텍스트)
3. 탭 레이블이 있으므로 패널 본문의 "회의록" / "스크립트" 제목(`h2`)은 제거한다.

### 공통

1. 버튼은 공유 컴포넌트를 사용해 스타일·동작을 일관되게 한다.
2. 클릭 가능한 버튼·탭에는 `cursor-pointer`를 적용한다.

## 비기능

- 접근성: 기존 `aria-label` 수준 유지 또는 보강.
- `prefers-reduced-motion`이 있으면 애니메이션을 최소화한다.

## 의존성

- `lucide-react` 패키지 추가.
