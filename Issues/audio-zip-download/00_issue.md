# 세션 오디오: ZIP 단일 다운로드

## 배경

세션 상세에서 오디오 세그먼트가 여러 개일 때, 다운로드 버튼이 WebM 파일을 n개 연속으로 내려받아 브라우저·OS 동작이 거칠다.

## 요구사항

1. **ZIP 패키징**: 동일 세션의 오디오 세그먼트를 하나의 zip 파일로 내려받는다.
2. **내부 파일명**: 세그먼트 순서가 보존되도록 `…-segment-001.webm` 형식을 유지한다.
3. **단일 오디오 병합**: WebM 세그먼트를 바이트로 이어 붙이는 방식은 이번 범위에서 하지 않는다.
4. **UI**: zip 다운로드임을 버튼 문구·접근성 이름으로 명확히 한다.

## 완료 조건

- 클릭 한 번으로 zip 파일 하나만 다운로드된다.
- IndexedDB `session-audio` 저장 구조는 변경하지 않는다.
- Vitest 및 품질 게이트를 통과한다.
