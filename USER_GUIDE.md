# Whirr 사용 방법

Whirr는 회의나 대화를 녹음한 뒤, 내용을 글로 바꾸고 AI 요약을 만들어 주는 회의 노트 앱입니다.

녹음 기록은 사용 중인 브라우저 안에 저장됩니다. 같은 컴퓨터라도 다른 브라우저를 쓰거나, 다른 기기에서 열면 이전 기록이 보이지 않을 수 있습니다.

## 시작하기 전에

- 마이크를 사용할 수 있는 데스크톱 브라우저에서 여는 것을 권장합니다.
- 처음 녹음을 시작할 때 브라우저가 마이크 사용 권한을 물어보면 허용해야 합니다.
- 녹음 내용을 글로 바꾸고 요약하려면 인터넷 연결이 필요합니다.
- 로그인은 없습니다. 기록은 현재 브라우저에만 남습니다.

## 첫 화면

앱을 열면 가운데에 노트 작성 영역이 보이고, 왼쪽에는 `History`가 보입니다.

노트 작성 영역에서는 다음 내용을 미리 적을 수 있습니다.

- 노트 제목
- 참석자
- 주제
- 키워드
- 요약 형식

이 항목들은 모두 선택 사항입니다. 비워 두고 바로 녹음해도 됩니다. 다만 회의 정보를 미리 적어 두면 요약이 더 알맞게 만들어집니다.

## 녹음하기

1. 필요하면 노트 제목, 참석자, 주제, 키워드, 요약 형식을 입력합니다.
2. 화면 아래쪽 녹음 카드에서 시작 버튼을 누릅니다.
3. 브라우저가 마이크 권한을 물어보면 허용합니다.
4. 회의나 대화를 진행합니다.
5. 녹음을 끝내려면 종료 버튼을 누릅니다.

녹음 중에는 경과 시간과 마이크 소리 크기를 볼 수 있습니다. 녹음이 끝나면 Whirr가 자동으로 내용을 글로 바꾸고 AI 요약을 만듭니다.

처리가 진행 중일 때는 새 녹음을 바로 시작할 수 없습니다. 잠시 기다린 뒤 다시 시작하세요.

## 요약 형식 고르기

요약 형식은 AI 요약이 어떤 모양으로 정리될지 정하는 항목입니다.

현재 고를 수 있는 형식은 다음과 같습니다.

- 기본회의: 논의 내용, 결정 사항, 해야 할 일 중심으로 정리합니다.
- 정보전달: 강연, 교육, 발표처럼 정보를 전달하는 내용을 학습 노트처럼 정리합니다.
- 비즈니스 미팅: 상대의 요구, 조건, 합의, 위험 요소 중심으로 정리합니다.
- 직접입력: 원하는 목차나 정리 방식을 직접 적습니다.

`AI 요약` 탭에서는 선택한 요약 형식의 예시 구조를 볼 수 있습니다. 실제 요약 내용은 녹음이 끝난 뒤 만들어집니다.

## 스크립트와 AI 요약 보기

녹음이 끝나고 처리가 완료되면 완료 알림이 나타납니다. 알림의 바로 보기를 누르거나, 왼쪽 `History`에서 기록을 선택해 상세 화면으로 이동할 수 있습니다.

상세 화면에는 두 가지 탭이 있습니다.

- AI 요약: 만들어진 회의 요약을 봅니다.
- 스크립트: 녹음 내용을 글로 바꾼 본문을 봅니다.

요약과 스크립트는 각각 복사 버튼으로 복사할 수 있습니다.

## 지난 기록 다시 열기

왼쪽 `History`에는 저장된 녹음 기록이 날짜별로 표시됩니다.

- 노트 제목이 있으면 제목이 먼저 보입니다.
- 제목이 없으면 스크립트 일부가 미리 보입니다.
- 현재 보고 있는 기록은 목록에서 강조됩니다.

화면 폭이 좁으면 `History`가 숨겨질 수 있습니다. 이때는 상단 메뉴 버튼을 눌러 열 수 있습니다.

녹음 중에는 다른 기록으로 이동할 수 없습니다. 녹음을 끝낸 뒤 이동하세요.

## 기록 편집하기

상세 화면의 편집 버튼을 누르면 저장된 내용을 고칠 수 있습니다.

편집 창에서 바꿀 수 있는 내용은 다음과 같습니다.

- 노트 제목
- 참석자
- 주제
- 키워드
- 요약 형식
- 스크립트 본문

수정한 뒤 저장 버튼을 누르면 현재 기록에 반영됩니다.

저장하지 않은 변경이 있을 때 편집 창을 닫으려 하면 확인 메시지가 나타납니다.

## AI 요약 다시 만들기

스크립트나 회의 정보를 고친 뒤 요약을 다시 만들 수 있습니다.

편집 창 아래쪽에서 다음 중 하나를 선택합니다.

- 현재 세션에 요약 재생성: 지금 보고 있는 기록의 요약을 새 요약으로 바꿉니다.
- 새 세션에 요약 재생성: 원래 기록은 그대로 두고, 새 기록으로 요약을 만듭니다.

현재 기록에 다시 만들기를 선택하면 기존 요약이 바뀌므로 확인 메시지가 먼저 나타납니다.

스크립트가 비어 있으면 요약을 만들 수 없습니다.

## 오디오 내려받기

상세 화면에 오디오 다운로드 버튼이 보이면 녹음 파일을 ZIP 파일로 내려받을 수 있습니다.

오디오가 남아 있지 않은 기록에는 다운로드 버튼이 보이지 않습니다.

## 기록이 저장되는 곳

Whirr의 기록은 앱 서버가 아니라 현재 브라우저 안에 저장됩니다.

그래서 다음 사항을 기억해 주세요.

- 다른 브라우저나 다른 기기에서는 기록이 보이지 않습니다.
- 브라우저 데이터를 지우면 기록이 사라질 수 있습니다.
- 현재 앱에는 기록 삭제, 검색, 공유 기능이 없습니다.
- 앱을 닫기 전에 녹음이나 요약 만들기가 끝났는지 확인하는 것이 좋습니다.

## 문제가 생겼을 때

- 마이크가 작동하지 않으면 브라우저의 마이크 권한을 확인하세요.
- 녹음 직후 글 변환이나 요약이 진행 중이면 잠시 기다리세요.
- 요약 만들기에 실패하면 상세 화면의 편집에서 다시 시도할 수 있습니다.
- 저장된 기록이 보이지 않으면 같은 브라우저에서 열었는지 확인하세요.
- 처리 중에 탭을 닫거나 새로고침하려 하면 브라우저가 확인 메시지를 보여 줄 수 있습니다.
